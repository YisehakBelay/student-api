# Student API 

A REST API built with Node.js, Express, and MongoDB.

## Features
- Create student
- Get all students
- Get student by ID
- Update student
- Delete student

## Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)

## API Endpoints

| Method | Endpoint |
|--------|--------|
| GET | /api/students |
| GET | /api/students/:id |
| POST | /api/students |
| PUT | /api/students/:id |
| DELETE | /api/students/:id |

## Run Locally

```bash
npm install
node server.js
